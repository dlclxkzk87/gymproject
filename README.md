## 프로젝트 기간
- **2025.03.01 ~ 2025.04.09** (총 5주)

## 프로젝트 개요
GYM management은 헬스장에서 회원과 관리자가 하나의 프로그램으로
각각의 개인정보 및 헬스장 관리를 일원화 시킨 **CUI기반 Java 어플리케이션** 입니다.
